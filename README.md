# 智能薪资计算器 (Salary Calculator)

一个基于 React + TypeScript + Vite 的智能薪资计算器，支持详细的个税计算、五险一金、加班费、年终奖及专项附加扣除。

## 功能特性

### 薪资计算
- **月度薪资明细**：计算12个月的详细薪资数据
- **五险一金**：自动计算养老、医疗、失业保险及公积金
- **个税累计预扣法**：按照国家规定累计预扣法计算个人所得税
- **加班费计算**：支持固定加班日（周六/周日），可配置加班倍数
- **年终奖单独计税**：年终奖按全年一次性奖金政策独立计税

### 专项附加扣除
支持以下8项专项附加扣除：
- 子女教育
- 继续教育
- 大病医疗
- 住房贷款利息
- 住房租金
- 赡养老人
- 个人养老金
- 税优健康险

### 数据可视化
- 月度薪资构成堆叠面积图
- 每月收入构成堆叠柱状图
- 全年收入对比柱状图
- 年度汇总卡片展示

### 其他特性
- 本地存储：自动保存计算结果，下次打开自动恢复
- 响应式设计：适配桌面和移动设备
- 详细说明：每项数据都有清晰的图例和温馨提示

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 6
- **样式**：TailwindCSS
- **图表**：ECharts + echarts-for-react
- **图标**：Lucide React

## 计算规则说明

### 个税计算
- 采用累计预扣法，当月税额 = 累计应缴税额 - 已缴税额
- 起征点：5000元/月
- 税率表遵循国家最新标准

### 五险一金（个人部分）
- 养老保险：8%
- 医疗保险：2%
- 失业保险：0.5%
- 公积金：5%-12%（可自定义）

### 加班费
- 计算公式：日均薪资 × 加班天数 × 加班倍数
- 日均薪资 = 基本薪资 ÷ 当月工作日

### 年终奖
- 单独计税，不计入综合所得累计计算

## 声明

本工具计算结果仅供参考，实际薪资以单位发放为准。不同地区五险一金缴纳基数和比例可能有所不同。
